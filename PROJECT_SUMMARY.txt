JONATHAN PAUL IVE (Design Principles)
  ✅ Obsessive Simplicity
     - Removed decorative elements
     - Clear, focused hierarchy
     - Whitespace as design element
  
  ✅ Clarity Through Reduction
     - One font family (Playfair + Source Sans 3)
     - One color system (primary blue + neutrals)
     - Clean, uncluttered layout
  
  ✅ Purposeful Material
     - Every pixel serves a purpose
     - Professional imagery approach
     - Consistent typography scale
  
  ✅ Elegant Constraints
     - Limited color palette
     - Consistent grid system
     - No unnecessary animations

ANDREJ KARPATHY (Development Best Practices)
  ✅ State Assumptions Explicitly
     - Clear information architecture
     - Obvious user journey
     - Transparent navigation
  
  ✅ Surgical Code Changes
     - Consolidated from 30 JS files to 1
     - Consolidated from 6 CSS files to 1
     - Removed all bloat
  
  ✅ Verification Loops
     - Accessibility testing checkpoints
     - Performance audits (Lighthouse)
     - Cross-browser compatibility
  
  ✅ Restraint in Implementation
     - No forms (static inquiry only)
     - No animations (only functional)
     - Minimal JavaScript (vanilla only)
     - No heavy frameworks
